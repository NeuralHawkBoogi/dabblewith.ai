IT group/admin permission ask

Hi — I’m planning a small free practical AI learning pilot for Casagrand First City residents. Can I post one message in the IT opportunities group to check interest? The idea is resident-led and non-salesy: people can register interest for a free clubhouse/online session where we build useful AI workflows — resume/interview prep, office automation, coding assistants, founder workflows, student projects, or WhatsApp/community bots. I’ll keep it to one short post + one follow-up poll only if people respond. No sensitive data collection, no spam, and I’ll share aggregate interest back if useful. If okay, I’ll post the draft after your approval.

If admin asks “what exactly?”

It’s a free practical AI session pilot for residents. What residents get: - learn useful AI workflows by doing, not theory - bring one real work/study/business problem - leave with a practical workflow idea or starter template What I’ll share: - one interest post - one poll only if there is interest - optional clubhouse session if enough residents opt in What I will not do: - no paid pitch in the group - no scraping group members - no sensitive data collection - no repeated broadcasts
