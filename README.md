# dabblewith.ai

Static community site and operating notes for dabblewith.ai.

## Contents

- `index.html` — homepage
- `autopilot/` — AI-operated community OS page
- `sessions/` — session queue page
- `blog/` — generated SEO blog pages
- `privacy/`, `terms/` — legal pages
- `events/` — event planning docs
- `tickets/` — product/build tickets
- `scripts/` — static generation scripts
- `data/` — source data for generated pages

## Common commands

```bash
node scripts/run-community-autopilot.js
node scripts/generate-blog.js
```
