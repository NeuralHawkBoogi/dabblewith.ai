privacy rule

Keep exports aggregated.

Export daily totals by event, page path, CTA id, audience segment, workflow category, UTM, and intent. Do not export WhatsApp text, email addresses, phone numbers, raw URLs, user IDs, or IP/device identifiers.

weekly operating loop

Ship → measure → decide.

  1. Export GA event totals into the CSV template.
  2. Mark the top converting surfaces and dead surfaces.
  3. Promote what creates replies, worksheet copies, submissions, or bot setup conversations.
  4. Archive vanity experiments that only create traffic.
