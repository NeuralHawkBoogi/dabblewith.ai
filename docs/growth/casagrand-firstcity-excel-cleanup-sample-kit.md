## Demo outline

1. Start with a 5–10 row fake sheet.
2. Normalize dates, owner names, status labels, and amount fields.
3. Flag likely duplicates and missing values.
4. Produce a human-review summary and follow-up questions.
5. End with a reusable prompt/checklist the resident can apply to the next sheet.
