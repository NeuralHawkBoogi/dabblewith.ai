Use these values in Google Cloud Console for the dabblewith.ai OAuth client. This page is intentionally simple and safe to leave public.

Where this is used

  • Google Analytics Data API authorization for pulling dabblewith.ai GA4 reports.
  • Google Analytics Admin API access for property discovery when needed.
  • OAuth re-auth using an account that has access to GA4 property 538311897.

Start page: /oauth/login/. Callback page: /oauth/callback/. The callback only displays OAuth query/hash values locally in the browser and does not transmit them anywhere.
